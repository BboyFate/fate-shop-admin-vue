# 商城后台
前后端分离的商城后台管理

## 本地安装访问

```bash
# clone the project
git clone https://github.com/BboyFate/fate-shop-admin-vue.git

# enter the project directory
cd fate-shop-admin-vue

# install dependency
npm install

# develop
npm run dev
```

访问后台 http://localhost:8080
